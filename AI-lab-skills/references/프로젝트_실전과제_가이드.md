# 프로젝트: 실전 과제 5종 가이드

## 한눈에 보기

5일간 배운 조각을 **오후 4시간에 하나로 조립**하는 팀 프로젝트입니다. 목표는 완제품이 아니라 **PoC(개념검증)** — "돌아가는 것 하나".
프로젝트가 무엇이든 **공통 안전장치 3종**(근거 표시 / 정직한 실패 / 멈추는 장치)이 없으면 금융권에서는 쓸 수 없는 물건이라는 것이 이 과제의 핵심 메시지입니다.

## 핵심 개념

### 공통 안전장치 3종 ⭐ (모든 프로젝트 필수)

| # | 안전장치 | 왜 | 배운 곳 |
|---|---|---|---|
| ① | **근거 표시** — 답의 출처(문서·조항·SQL·티켓)를 화면에 함께 | 검증할 수 없는 답은 금융에서 쓸 수 없음 | [[개념_RAG_파이프라인]], [[개념_문서유형별_전처리]] |
| ② | **정직한 실패** — 모르면 "확인되지 않습니다" + 담당자 안내 | 지어낸 답 1건이 서비스를 죽임 | [[개념_Agent_통제력과_루프탈출]] |
| ③ | **멈추는 장치** — 루프 상한 / 사람 승인 / 권한 차단 중 최소 1개 | 통제 없는 Agent는 사고 | [[개념_HITL과_검증루프]] |

그리고 **실행 경로를 화면에 보여주는 것** — 어떤 노드를 거쳤는지, 어떤 도구를 썼는지. 시연에서 반응이 가장 좋은 부분입니다.

### 프로젝트 5종

| # | 프로젝트 | 대상 부서 | 핵심 기술 | 난이도 | 관련 위키 |
|---|---|---|---|---|---|

| 3 | **사내데이터 분석봇** | 마케팅·경영관리·DB | Text-to-SQL + **권한 가드레일** | ⭐⭐ | [[개념_Text_to_SQL]] |

## 핵심 코드 — 공통 파일 3개

팀 폴더에 그대로 복사해 쓰는 재사용 모듈입니다.

### 1. `common/llm_provider.py` — 가장 중요

모든 LLM 호출이 여기를 거치게 하면 OpenAI ↔ Ollama를 **환경변수 하나로** 전환할 수 있습니다.

```python
import os
BACKEND = os.environ.get("LLM_BACKEND", "openai")   # openai | ollama

def get_llm(temperature: float = 0.0):
    if BACKEND == "ollama":
        from langchain_ollama import ChatOllama
        return ChatOllama(model=os.environ.get("LLM_MODEL", "qwen2.5:7b"), temperature=temperature)
    from langchain_openai import ChatOpenAI
    return ChatOpenAI(model=os.environ.get("LLM_MODEL", "gpt-4.1"), temperature=temperature)

def get_embeddings():
    if BACKEND == "ollama":
        from langchain_ollama import OllamaEmbeddings
        return OllamaEmbeddings(model="bge-m3")
    from langchain_openai import OpenAIEmbeddings
    return OpenAIEmbeddings(model="text-embedding-3-small")
```

> ⚠️ **임베딩을 바꾸면 벡터DB를 다시 만들어야 합니다.** 오늘은 하나로 정하고 끝까지 가세요.

### 2. `common/pii.py` — 마스킹 (프로젝트 2·3 필수)

```python
import re
PATTERNS = {
    "주민번호": r"\d{6}[-\s]?[1-4]\d{6}",
    "카드번호": r"\d{4}[-\s]?\d{4}[-\s]?\d{4}[-\s]?\d{4}",
    "계좌번호": r"\d{3,6}[-\s]\d{2,6}[-\s]\d{2,6}",
    "휴대폰":   r"01[016-9][-\s]?\d{3,4}[-\s]?\d{4}",
}
def mask(text: str) -> str:
    for label, pat in PATTERNS.items():
        text = re.sub(pat, f"[{label}]", text)
    return text
```

### 3. `common/audit.py` — 감사 로그

```python
import json, datetime, pathlib
LOG = pathlib.Path("./logs/audit.jsonl")

def log(**kw):
    LOG.parent.mkdir(exist_ok=True)
    kw["ts"] = datetime.datetime.now().isoformat(timespec="seconds")
    with LOG.open("a", encoding="utf-8") as f:
        f.write(json.dumps(kw, ensure_ascii=False) + "\n")

# 쓰는 법 — 질문·답변은 반드시 마스킹
log(user="김대리", question=mask(q), sources=[...], answer=mask(a), path=["classify", "retrieve"])
```

> 데모에서 이 파일을 열어 보여주면 "감사 대응이 되네"라는 반응이 나옵니다. 30초 투자로 효과가 큽니다.

### 4. 프로젝트 1 예시 — MUST 3개와 도전 과제

| # | 기능 | 확인 방법 |
|---|---|---|
| M1 | 문서 3종(약관/FAQ/금리표)을 **유형별로 다르게 전처리** | 약관은 조항 단위, FAQ는 Q&A 쌍, 표는 행→문장 |
| M2 | 모든 답변에 **근거 조항명 + 원문 스니펫** | 답변 5개 전부 출처가 붙는가 |
| M3 | 근거를 못 찾으면 **"확인되지 않습니다" + 담당부서 안내** | 없는 규정을 물었을 때 지어내지 않는가 |

도전 과제 중 **인용 검증 노드**가 특히 실무적입니다:

> "답변이 인용한 조항 번호가 검색 문서에 **실제로 있는지 문자열로 대조**하세요.
> **LLM에게 물어보지 마세요. 지어낸 조항을 '있다'고 답합니다.**"

프로젝트 1의 범위 선 긋기도 중요합니다:

> ❗ **이 봇은 "대출 가능/불가"를 판정하지 않습니다.** 판단에 필요한 **근거를 찾아주는** 역할까지입니다.

## 5분 데모 구성

발표가 아니라 **시연**입니다. 슬라이드 금지.

```
0:00–0:30  "우리 부서는 이런 문제가 있습니다"        ← 딱 한 문장
0:30–2:00  정상 시나리오 실행 — 잘 되는 장면
2:00–3:30  ★ 안전장치 시연 — 모를 때 / 권한 없을 때 / 승인 없이 못 나갈 때
3:30–4:30  "실제로 도입하려면 뭐가 더 필요한가"
4:30–5:00  질의응답
```

> ⭐ **2:00–3:30이 하이라이트입니다.** 잘 되는 장면은 누구나 보여줍니다. **"모른다고 답하는 장면"** 을 보여주는 팀이 이깁니다.

## 오늘의 규칙 4가지

1. **API 키 하드코딩 금지** — `os.environ.get(...)` 패턴 유지
2. **Mock 데이터는 주석에 `(Mock)` 표기** — 시연 중 "실제 데이터냐" 질문이 반드시 나옴
3. **단일 파일 완결** — `app.py` 하나. 모듈 쪼개다 시간 감
4. **주석은 한국어**, 상단에 `# 실행: streamlit run app.py`

## 주의사항

- 🚨 **실제 고객 데이터는 절대 쓰지 마세요.** 전부 가공 데이터입니다.
- **샘플 데이터는 직접 만들지 말고 클로드코드에게 시키세요.** 각 프로젝트 STEP 0에 데이터 생성 프롬프트가 있습니다. 5분이면 끝날 일을 손으로 하면 1시간이 날아갑니다.
- **완벽하게 만들려다 아무것도 못 보여주는 게 최악입니다.** 목표는 "돌아가는 것 하나".
- **프로젝트 4(멀티에이전트)는 시간이 빠듯합니다.** Supervisor가 FINISH를 못 고르면 무한 루프이므로 `recursion_limit` 설정이 필수입니다 → [[개념_MultiAgent_Supervisor]]
- **프로젝트 3(Text-to-SQL)은 권한 가드레일이 MUST**입니다. SELECT만 허용 + 읽기 전용 계정 → [[개념_Text_to_SQL]]
- **드라이버는 1시간마다 교대하세요.** 클로드코드를 직접 몰아보는 게 가장 큰 학습입니다.
